# Flowers Frontend

A simple static frontend project for a flower-themed website.

## Features

- Clean HTML structure in `index.html`
- Centralized styling in `style.css`
- Lightweight and easy to customize

## Project Structure

```
Flowers-frontend/
├── index.html
├── style.css
└── README.md
```

## Getting Started

Since this is a static frontend project, no installation is required.

1. Open the project folder.
2. Open `index.html` in your browser.

### Run with VS Code Live Server (optional)

If you use VS Code, you can run the project with the Live Server extension:

1. Install the **Live Server** extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

## Customization

- Update page content in `index.html`.
- Change colors, spacing, and layout in `style.css`.
